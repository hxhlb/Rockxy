title: "Keyboard Shortcuts"
description: "Current keyboard shortcuts for proxy control, sessions, tabs, navigation, rules, diff, and scripting"
---

Rockxy supports keyboard shortcuts for the most common debugging actions. All shortcuts are accessible from the menu bar as well.

## Proxy Control

| Shortcut | Action |
|----------|--------|
| `Cmd + Shift + R` | Start Proxy |
| `Cmd + .` | Stop Proxy |
| `Cmd + Shift + D` | Toggle Recording |

## Tabs and Sessions

| Shortcut | Action |
|----------|--------|
| `Cmd + T` | New workspace tab |
| `Cmd + W` | Close workspace tab |
| `Cmd + Shift + R` | Rename workspace tab |
| `Cmd + Shift + N` | New Session |
| `Cmd + O` | Open Session |
| `Cmd + Shift + S` | Save Session |
| `Cmd + Shift + I` | Import HAR |
| `Cmd + Shift + E` | Export HAR |
| `Cmd + Option + Shift + Delete` | Clear Session |

## Flow and Selection

| Shortcut | Action |
|----------|--------|
| `Cmd + Return` | Repeat selected request |
| `Cmd + Option + Return` | Edit and Repeat |
| `Cmd + C` | Copy selected URL |
| `Cmd + Shift + C` | Copy selected request as cURL |
| `Delete` | Delete selected request |
| `Cmd + Shift + Delete` | Delete all requests |

## Navigation and Tools

| Shortcut | Action |
|----------|--------|
| `Cmd + Shift + F` | Filter Domain or App |
| `Cmd + L` | Focus URL search / add comment, depending on active menu context |
| `Cmd + Shift + L` | Auto Select the Latest Request |
| `Cmd + Control + [` | Toggle Source List Panel |
| `Cmd + Control + ]` | Dock inspector to bottom |
| `Cmd + Control + \\` | Dock inspector to right |
| `Cmd + Shift + ]` | Select next workspace tab |
| `Cmd + Shift + [` | Select previous workspace tab |
| `Cmd + Option + P` | SSL Proxying List |
| `Cmd + Option + B` | Bypass Proxy List |
| `Cmd + Shift + B` | Breakpoint Rules |
| `Cmd + Option + L` | Map Local |
| `Cmd + Option + R` | Map Remote |
| `Cmd + Option + [` | Block List |
| `Cmd + Option + A` | Allow List |
| `Cmd + Option + N` | Network Conditions |
| `Cmd + Option + Y` | Open Diff View |
| `Cmd + Option + D` | Compare Selected |
| `Cmd + Option + I` | Script List |

## General

| Shortcut | Action |
|----------|--------|
| `Cmd + ,` | Open Settings |

<Tip>
  Check the menu bar for the full list of available shortcuts. Some actions gain keyboard shortcuts when plugins are installed.
</Tip>
